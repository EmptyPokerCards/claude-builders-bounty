# 🤖 Claude PR Review Agent (Zero Dependencies)

A blazingly fast, zero-dependency Python CLI tool that harnesses Anthropic's Claude 3.5 Sonnet to automatically review GitHub Pull Requests. It analyzes unified diffs and outputs strict, beautifully formatted Markdown conforming precisely to the requested acceptance criteria.

## 🚀 Features

- **Zero Third-Party Dependencies:** Written in pure Python 3 using standard libraries (`urllib`, `json`, `argparse`). No need to run `pip install` or install the `gh` CLI tool!
- **Direct GitHub API Integration:** Fetches the raw unified diff directly from GitHub instantly.
- **Strict Markdown Compliance:** Uses highly-engineered System Prompts to guarantee Claude returns the exact structure required: `Summary`, `Risks`, `Suggestions`, and `Confidence score`. (We do NOT output JSON, as Markdown is the universally requested standard for GitHub comments).
- **Safety Bounds:** Protects against token limits by automatically truncating astronomically large PR diffs before analysis.

## 💻 Installation

Since there are no dependencies, installation is just one command:

```bash
curl -O https://raw.githubusercontent.com/owner/repo/main/claude-review.py
chmod +x claude-review.py
```

## 🛠 Usage

1. Export your Anthropic API Key:
```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

2. Run the tool against any open PR:
```bash
./claude-review.py --pr https://github.com/owner/repo/pull/123
```

## How it works

1. Parses the GitHub PR URL.
2. Calls `https://api.github.com/repos/{owner}/{repo}/pulls/{number}` with the `application/vnd.github.v3.diff` header.
3. Formats the diff into a secure payload and queries `https://api.anthropic.com/v1/messages`.
4. Outputs the Markdown directly to `stdout`.
